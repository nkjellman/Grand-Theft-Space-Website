Download Grand Theft Space

Description
This is a GTA 5 expansion that allows you to traverse outside of the world of GTA, into the solar system and beyond! This mod was not made to be a mission based mod, but more of a platform for experienced developers to integrate scripts / maps / and other things into the DLC in order to create content and expand the community experience.

Standard

INFO

This is the main Grand Theft Space package. It is designed to run with vanilla GTA V graphics. To use with any other graphics mod, be sure to install an Add-On package for your graphics mod.

REQUIREMENTS
  • Download and install NoBoundaryLimits.
  • Download and install .NET Framework 4.5.2
  • Download and install C++ Redist 2017
  • Download and install the latest ScriptHookV, and ScriptHookVDotNet files.
  • Download and install the latest NativeUI release.
INSTALL & USE
  • Install the mod files using OpenIV's package installer.
  • Once in-game you will see a marker displayed on your map indicating you need to do the first mission in order to travel into the stars!
  • If anything goes wrong, or you're just not patient, you can always use Numpad9 to open a menu, and select a scene you wish to travel to from there.

VisualV Add-On

INFO

This is the Grand Theft Space VisualV Add On which will make your interstellar adventures through the cosmos look even more stunning.

INSTALL & USE

Make sure you install the requirements in the order listed below.

  1. Download and install VisualV.
  2. Install the Earth Atmosphere add on that came with VisualV.
  3. Download and install the GTS Standard Package and all of its requirements.
  4. Install the Add-On using the Open IV Package Installer
NOTE

Compatibility between GTS and the atmosphere script has not been added yet. So for the two mods to work together, you should press PG UP right before going into space to disable the atmosphere script, then press PG UP when returning to Earth to reactivate it. In the future, we are hoping to have it be compatible with the mod, and have the atmosphere effects on other planets besides Earth.
